## Summary

This document provides a complete reference of all colors used throughout the BookFinder app. All colors are defined in RGBA format with explicit opacity values, making it easy to:

1. Maintain consistent colors across all screens
2. Implement proper light/dark mode support
3. Follow Material Design 3 guidelines
4. Ensure accessibility with proper contrast ratios
5. Create new screens or components with consistent styling

For implementation, all these colors are defined as CSS custom properties in `/styles/globals.css` and can be accessed using Tailwind CSS utility classes or the `var(--color-name)` syntax.
